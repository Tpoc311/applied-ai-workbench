# Applied AI Workbench

A personal workbench for practical AI/ML/CV projects — experiments, prototypes, task-focused implementations, and
reusable components.

## About

This repository is my personal workbench for hands-on practice across applied AI tasks, with a strong focus on computer
vision.

It is used to organize experiments, document problem-solving approaches, and gradually build a base of practical
implementations for future projects and real-world work.

## Scope

The repository may include work related to:

- computer vision
- machine learning
- deep learning
- applied AI experiments and prototypes
